# 🧮 Modern Python Calculator

A modern desktop calculator application built using **Python** and **Tkinter**. This project features a clean graphical user interface and supports basic arithmetic operations. It was created as part of my programming journey while learning Python and building real-world applications.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔢 Decimal support
- 🧹 Clear button
- ⌫ Delete (Backspace)
- 🖥️ Modern graphical user interface
- ⚡ Instant calculations
- 🚫 Basic error handling

## 🚀 How to Run

1. Install Python 3
2. Clone this repository

```bash
git clone https://github.com/pranit-noaim/modern-python-calculator.git
```

3. Navigate into the project

```bash
cd modern-python-calculator
```

4. Run the application

```bash
python calculator.py
```
